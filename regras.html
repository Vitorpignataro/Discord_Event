Regras
  • Regra 1#

    O projeto deve conter somente HTML, CSS e JavaScript e deve ser responsivo para todas as plataformas. Não é permitido o uso de frameworks neste desafio.

  • Regra 2#

    Os critérios de avaliação dos jurados serão: design, criatividade, responsividade e user experience.

  • Regra 3#

    O tema da landing page é livre, porém é proibido qualquer tema que possa ser ofensivo, contenha um conteúdo gore ou seja desagradável.

  • Regra 4#

    O envio deve ser de forma anonima e zipado, para o drive https://bit.ly/3oNQvWq, só haverá a identificação do responsável após julgamento dos jurados.

  • Regra 5#

    A entrega deverá ser feita até dia 05/08/2022 no máximo até 23:59h. (CountDown será atualizado para a data de entrega).

  • Regra 6#

    O anúncio do vencedor será dia 07/08/2022, as 21:00h, será feito uma transmissão ao-vivo no Discord, será mostrado cada projeto e os critérios de avaliação.
